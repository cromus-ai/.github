---

## The Core Idea

Every organization has institutional knowledge — SOPs, runbooks, process documents — sitting in folders, never touched by AI because nobody compiled them into a format AI systems can actually use.

SKILL.md is that format.

A SKILL.md file captures what a skill does, how it does it, what it costs, what governs it, and where it runs — in plain text, readable by a developer, a compliance officer, and any AI framework simultaneously.

**Intelligence precedes execution. Always.**
